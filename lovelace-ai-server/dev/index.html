Como rodar o preview localmente

Abra um terminal na pasta do projeto e rode qualquer um dos servidores abaixo:

python -m http.server 8080

npx serve .

Depois acesse http://localhost:8080/dev/ no navegador.

Para instalar no HA: copie lovelace-ai-server.js para config/www/ e adicione como recurso externo /local/lovelace-ai-server.js.
