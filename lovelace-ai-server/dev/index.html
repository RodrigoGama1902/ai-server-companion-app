AI Server Card — Preview Local

Simula o card exatamente como aparece no Home Assistant

Configuração do Card (YAML)

type: custom:lovelace-ai-server
title: AI Server
server_entity: switch.ai_server
cpu_entity: sensor.ai_server_cpu_usage
gpu_entity: sensor.ai_server_gpu_usage
ram_entity: sensor.ai_server_ram_usage
vram_entity: sensor.ai_server_vram_usage
cost_entity: sensor.custo_diario_ai_machine
monthly_cost_entity: sensor.custo_mensal_ai_machine
Como rodar o preview localmente

Abra um terminal na pasta do projeto e rode qualquer um dos servidores abaixo:

python -m http.server 8080

npx serve .

Depois acesse http://localhost:8080/dev/ no navegador.

Para instalar no HA: copie lovelace-ai-server.js para config/www/ e adicione como recurso externo /local/lovelace-ai-server.js.
